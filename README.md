# Spring Boot Student Mini Project

## Description
Simple Spring Boot REST API project to manage students.
Functions:
- Add student
- View all students

## Technologies
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Endpoints
POST /students → Add student  
GET /students → View all students

## How to Run
1. Clone repository
2. Open in IntelliJ
3. Run StudentappApplication.java
4. Access: http://localhost:8080/students
